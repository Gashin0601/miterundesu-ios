# ミテルンデス (Miterundesu)

**撮影じゃない、拡大して見るためのアプリ** — 美術館・博物館・劇場で安心して使える、プライバシー重視の拡大鏡。

[![Swift](https://img.shields.io/badge/Swift-5.0-orange)](https://swift.org)
[![iOS](https://img.shields.io/badge/iOS-17.0+-blue)](https://developer.apple.com/ios/)

## 概要

視覚障害者や高齢者が文化施設で安心して使える拡大カメラアプリです。撮影した画像は端末内（Core Data）にのみ一時保存され、撮影から10分後に個別タイマーで自動削除されます。iOS の写真ライブラリには一切書き出しません。スクリーンショットや画面録画もブロックし、施設側も来場者も安心して使える仕組みを実現しています。

## 主な機能

- **高倍率ズーム** — ピンチ／±ボタン／iOS 18+ のカメラコントロールで拡大。デフォルト最大100倍、設定で最大200倍まで拡張可（デバイス上限に準拠）
- **1倍ボタン長押し** — 拡大中に1倍ボタンを長押しすると一時的に等倍表示、離すと元の倍率に復帰
- **10分自動削除** — 撮影画像は Core Data に一時保存され、個別タイマーで撮影から10分後に自動消去
- **スクリーンショット・画面録画防止** — 撮影はセキュアテキストフィールドでブロック、録画検出時はプレビューをぼかし表示
- **シアターモード** — 美術館・劇場向けに撮影機能を無効化、UIカラーも専用に切り替え
- **プレスモード** — 報道関係者向けの Supabase 認証モード
- **ウォーターマーク** — 保存画像にアプリ名・撮影日時・デバイスIDを永続的に焼き付け
- **クイック起動** — ロック画面ウィジェット、コントロールセンター（iOS 18+）、Siri・ショートカットから素早く起動
- **日本語 / 英語** — 設定から手動切り替え
- **VoiceOver対応** — 主要画面でローカライズ済みラベル・ヒント・カスタムトレイトを提供

## 技術スタック

- **言語:** Swift 5.0
- **UI:** SwiftUI
- **カメラ:** AVFoundation
- **データ:** Core Data
- **ウィジェット / インテント:** WidgetKit + AppIntents
- **認証（プレスモード専用）:** Supabase
- **対応OS:** iOS 17.0+
